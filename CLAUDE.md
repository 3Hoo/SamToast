# 목적

클로드 코드의 작업 완료 또는 유저의 인풋/승인 대기 등을 좀 더 직관적으로 알리는 앱의 개발.  
다만 커스텀 가능한 요소를 최대한 제공하여, 개발자가 재미있게 자신의 개발 환경을 구성할 수 있도록.

# 기술 스택

tauri + rust

# 목표 기능

클로드 코드에 hook을 걸어 유저에게 알림을 주는 시스템

## 알림 관련

- 알림은 화면 아무 곳이나 배치할 수 있으며 숨길 수도 있음 (숨길 경우 hook 발생시 다시 발생)
- 알림은 클로드 코드 세션당 하나씩 개별적으로
- timeout 되었을 때 자동으로 숨김 처리 된다
- 알림 이미지 커스텀 가능
  - 만약 이미지 경로에 0부터 시작하는 이미지들이 있다면 애니메이션으로 재생

## 설정 gui 관련

트레이를 좌클릭하면 설정 gui가 뜬다. 설정 gui에서는 다음을 설정할 수 있다

- hook 대상 이벤트
  - 각 hook 대상 이벤트마다 다음 설정이 가능하다
    - hook 발생 알림음 경로 (default는 윈도우 알림음)
    - hook 발생 알림 이미지 경로 (default는 claude 아이콘이 있는 일반적인 토스트 알림)
- 알림 클릭 반응 설정
  - timeout 설정
    - 0초면 timeout 없음
  - 클릭시 이동 설정
    - 대상 클로드 세션 이동
    - 아무런 반응 없음
  - 클릭시 알림 창 반응
    - 알림 창 닫기 (다시 대기 모드로)
    - 닫기 전용 이미지/애니메이션 출력
